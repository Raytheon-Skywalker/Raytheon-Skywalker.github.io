Magento 2 插件部署指南 (Based On Bitnami Magento Stack)

Bitnami Magento Stack具有多种的部署模式, 相关的操作都大同小异, 可以参照引用的官方文档中的内容操作

获取Bitnami的秘钥信息#Obtain SSH credentials from the Bitnami Launchpad

下载相应的PEM秘钥文件

登录到Magento 2 ‘s Server (Based on Bitnami Stack)

# 登录到服务器
sudo ssh -i #{秘钥名称}.pem bitnami@#{IP-Address}
# 查询Magento admin的default username and password密码
sudo cat /home/bitnami/bitnami_credentials

上传extension的Zip文件到服务器

sudo scp -i #{秘钥名称}.pem #{插件压缩包名称}.zip bitnami@#{IP-Address}:/opt/bitnami/stats

解压文件并转存到Magento安装目录

# 解压之后的目录结构是#{机构名称}/#{产品名称}/*
sudo unzip -o -d . #{插件压缩包名称}.zip
cd /opt/bitnami/magento
mkdir –p app/code/#{机构名称}/#{产品名称}/
cp -R #{压缩包解压路径}/#{机构名称}/#{产品名称}/* /opt/bitnami/magento/app/code/#{机构名称}/#{产品名称}/

执行Magento命令进行安装

cd /opt/bitnami/magento
chmod -R 777 var/ generated/ pub/static
sudo php bin/magento maintenance:enable
sudo php bin/magento setup:upgrade
sudo php bin/magento setup:di:compile
sudo php bin/magento setup:static-content:Deploy -f
sudo php bin/magento maintenance:disable
sudo /opt/bitnami/ctlscript.sh restart

使用Admin账号进行登录并进行插件相关的配置

Magento admin登录地址: http://IP-Address/admin

Magento日志位置

/opt/bitnami/magento/var/log
/opt/bitnami/magento/var/report
