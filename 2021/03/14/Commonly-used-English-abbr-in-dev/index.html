Git 团队协作中常用术语

  1. WIP   Work in progress, do not merge yet. // 开发中 传说中提 PR 的最佳实践是，如果你有个改动很大的 PR，可以在写了一部分的情况下先提交，但是在标题里写上 WIP，以告诉项目维护者这个功能还未完成，方便维护者提前 review 部分提交的代码。
  2. LGTM Looks good to me. // Riview 完别人的 PR ，没有问题
  3. SGTM: Sounds Good To Me. 和上面那句意思差不多，也是已经通过了 review 的意思
  4. PTAL Please take a look. // 帮我看下，一般都是请别人 review 自己的 PR
  5. TBR To Be Reviewed. 提示维护者进行 review
  6. CC Carbon copy // 一般代表抄送别人的意思
  7. RFC  —  request for comments. // 我觉得这个想法很好, 我们来一起讨论下
  8. IIRC  —  if I recall correctly. // 如果我没记错
  9. ACK  —  acknowledgement. // 我确认了或者我接受了,我承认了
  10. AFAIK / AFAICT ： 据我所知/可以说
  11. NACK/NAK — negative acknowledgement. // 我不同意
  12. PR: Pull Request. 拉取请求，给其他项目提交代码
  13. TL;DR: Too Long; Didn’t Read. 太长懒得看。也有很多文档在做简略描述之前会写这么一句
  14. TBD: To Be Done(or Defined/Discussed/Decided/Determined). 根据语境不同意义有所区别，但一般都是还没搞定的意思
  15. TBA (to be announced待公布)
  16. utACK：（又名。未经测试的确认， 同意这些更改并对其进行审核，但未进行测试）

办公室常用英语

一. 邮件中：

  1. BTW: by the way 顺便问下，顺便提下
  2. FYI: for your information 供参考
  3. FYR: for your reference 供参考
  4. EOD: end of the day 下班前，比如说EOD Thursday，就是周四下班前
  5. ASAP: as soon as possible 尽快
  6. TBD, TBC: to be determined/decided/confirmed 待定，比如会议室TBD，就是说会议室还待定的意思
  7. OT: overtime 加班
  8. P.S: postscript 补充说明
  9. RSVP: 法语Répondez s’il vous plait, 希望得到回复的时候加上
  10. CC: carbon copy 抄送，有时候同事和你讲，小A你邮件塞给我，指的就是邮件CC这个同事，用的谐音。
  11. BCC: blind carbon copy 隐秘抄送
  12. FW: forward 转发
  13. Re: reply 回复
  14. N/A: not applicable 表格中有时候不允许空值的话可以填这个
  15. Loop in / drop: 比如这封邮件，小A觉得和谁谁相关，就回复全部，加上loop in 谁谁，相当于@一样，drop的话也是觉得谁谁无关，就回复全部说drop谁谁，把那个人移除邮件会话。
  16. On annual leave/ business trip: 休年假，出差时往往也发封邮件给相关的同事告知。
  17. OOTO: out of the office, 外出不在办公室。
  18. AFAIK: as far as I know, 据我所知。
  19. BR: best regards, 祝好。
  20. BRB: Be Right Back 马上回来
  21. OOO: Out of Office=不上班，不在办公室I’m going to be OOO all of next week.
  22. NB: Nota Bene 特别注意，划重点 NB. You should read this post more than once. 划重点，这篇文章你得多读几遍。
  23. NRN: No Reply Necessary 不用回复 This is a work email, NRN. 这是工作邮件，不必回复。
  24. NSFW: Not Safe For Work 不适于工作场所 This attachment is NSFW，open it when you are out of the office. 这个附件不适合在办公室看，下班了再点开。
  25. CU: See you 再见
  26. CUL: See you later 下次再会
  27. OIC: oh I see
  28. Hand: Have a nice day 过得愉快
  29. Ty: thank you 谢谢你
  30. Np: no problem 没问题
  31. Yw: you are welcome 不客气
  32. Hru: How are you? 你好吗?
  33. ILU: I love you 我爱你
  34. Sup: What’s up 近来如何
  35. K/KK: ok,kk就像ok的轻松回答，更温柔,k虽然也表示OK，但会显得态度唐突
  36. Lol: laugh out loud 大笑,笑喷了
  37. Lmao: laugh my ass off 笑到屁股都掉了！
  38. Syl: See you later 等会儿见
  39. Wut/Wat: what,wut有种故意让自己变得呆萌的感觉,wat则有点想要可爱一点的感觉
  40. Ttyl: Talk to you later 待会儿再聊
  41. Rn: right now 现在
  42. ASAP: as soon as possible 尽快
  43. LTNS: long time no see 好久不见
  44. TBH: to be honest 老实说
  45. BID (break it down细分明细)
  46. COB (close of business下班时)
  47. EOB (end of business下班时)
  48. DOE (depending on experience视工作经验而定)
  49. ETA (estimated time of arrival预计到达时间)
  50. FWIW (for what it’s worth无论如何、不管有没有用)
  51. IAM (in a meeting会议中)
  52. IMO (in my opinion个人浅见、在我看来)
  53. LET (leaving early today今天提早走)
  54. LMK (let me know让我知道、通知我)
  55. POC (point of contact首要联系人)
  56. PTE (part-time employee兼职雇员)
  57. NDA (Non-disclosure agreement保密合约)
  58. RFD (request for discussion请求讨论)
  59. TCC (teleconference call电话会议)
  60. TED (Tell me, explain to me, describe to me)
  61. TLDR (Too long didn’t read太长，没读)
  62. AKA: as known as 也就是、以…为人所知
  63. AFK: away from keyboard 不在；不在位子上
  64. PFA: please find attached. 请参看附件
  65. LOI Letter of intent 意向书
  66. RFI request for information 信息请求
  67. RFP request for proposals 征求建议书

二. 开会时：

  1. 1:1: 念作one on one，指的是一对一开会，有时口语也会随口说one one。

  2. Keep it to yourself: 比如老板只告诉了你一个的信息，就会告诉你说，小A你自己知道就好了，就不要和别人讲了。

  3. Between us: 和上一条类似，只有我们两个人知道就好啦。

  4. Within the room: 开会时鼓励大家畅所欲言不要顾忌，就说今天讨论的都在这个屋子里，出了这个门就都不要说了，就忘记好了。

  5. Keep it confidential: 重要的材料或者会议的话，主持人都会提醒，这是公司内部机密，要注意保密之类的话。

  6. Compliance: 有时候会说要注意compliance方面的问题哦，就是指处理工作要合规，不要做不合规矩的事情。

  7. Echo: 比如会议上哪个同事发表了观点，小A就说我回应一下他的话，对他说的做一点补充，就可以说，我echo一下刚才他提到的那条，我也觉得blablabla.

  8. BS: brainstorming, 脑力激荡，大家脑洞大开发散思维讨论。

  9. Remote: 远程开会或办公。

  10. Work from home: 在家办公、开会。

  11. Keep you informed/updated: 有新的消息会随时通知大家。

  12. Q&A: question and ask, 提问环节，一般做完presentation会留出时间做这个部分。

  13. Orientation: 情况大致介绍。

    三. 和人相关：

  14. Candidate: 讨论某个项目的候选人。

  15. Owner: 指某个项目的负责人，比如老板问起某个项目，哪个项目谁负责的啊，就可能会说这个项目owner是谁？

  16. Direct report: 直线汇报，指的是顶头上司或者直线下属这一条直线。

  17. Line manager: 直线经理，指的是直接汇报给谁。

  18. People manager: 指实际带人的经历，而不是听上去职位头衔是经理但是不一定带人的，比如”产品经理”、”项目经理”等。

  19. Backup: 项目的备选负责人啦，休假时找个顶班的同事啦之类。

  20. Band / level: 通常用来指职级，在这个公司的职位级别，比如几级员工啦。

  21. VP: vice president，副总。

  22. VC: venture capital，风投。

  23. Background check: 入职前做的尽职调查。

  24. Onboard: 入职。

  25. Peer: 同级别的同事。

    四. 业务相关：

  26. ROI: return on investment, 投资回报率，比如说这个项目的ROI低啊，所以优先级比较低之类。

  27. SOP: standard operating procedure, 标准操作流程，比如一件事情完全

  28. Priority: 优先级，常常会听到有人说这个项目优先级不高。

  29. Keep an eye on it: 项目上线了可以和老板说，总之我会盯着的啦，您放心吧。

  30. CRM: customer Relationship Management, 客户关系管理。

  31. EDM: electric direct mail, 给客户发邮件的方式。

  32. FAQ: frequently asked questions, 常被问到的问题，一般新项目介绍最后会放一个FAQ文档，用于解答一般疑惑。

  33. OP: operation，运营。

  34. QA: quality assurance, 测试、检测。

  35. YOY: year over year, 年环比，比如说我们这个年环比增长40%，就可以说40% YOY.

  36. KPI: key performance indicator, 绩效。

  37. Scope: 指工作内容范围，比如小A的工作scope变大了，就是范围广了。

  38. Bandwidth: 带宽，这里指精力，比如和老板抱怨，老板，你给我这么多活，我的bandwidth不够啊。摊手。
