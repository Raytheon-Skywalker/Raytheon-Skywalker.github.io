主要是为了使用QGis加载不同图商的卫星地图进行相关数据的展示

  1. QGIS的安装

    1. 使用HomeBrew安装

      1. brew tap osgeo/osgeo4mac & brew install osgeo-qgis

  2. QGIS 加载在线地图资源

    1. 感谢同事丁总发给的在线地图资源配置URL

      卫片供应商Url
      Esrihttps://services.arcgisonline.com/arcgis/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}
      谷歌cnhttp://www.google.cn/maps/vt?lyrs=s&gl=cn&x={x}&y={y}&z={z}
      谷歌comhttps://www.google.com/maps/vt?lyrs=s&x={x}&y={y}&z={z}
      Mapboxhttps://d.tiles.mapbox.com/v4/digitalglobe.316c9a2e/{z}/{x}/{y}.png?access_token=XXXXXXX
      OSMhttp://tile.openstreetmap.org/{z}/{x}/{y}.png
      高德https://webst01.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}
