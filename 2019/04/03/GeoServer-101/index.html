    21. 1.0.21. Workspace
    22. 1.0.22. 像Namespace一样，是管理其他概念的容器。一般，相似的Layer放到同一个Workspace中，表达为WName:LayerName。相同LayerName的Layer需要放到不同的Worksapace中。
    23. 1.0.23. Stores
    24. 1.0.24. 链接了一个具体的数据源，可附加定制的链接参数，数据源可以是N个文件、一个文件夹或是一个数据库
    25. 1.0.25. Layers
    26. 1.0.26. 图层意味着一个栅格或是矢量的数据集。Layer依赖于Store中的具体数据，比如数据库中的某一个表，Layer被关联到Store所属的Workspace中。
    27. 1.0.27. Layer Groups
    28. 1.0.28. 是Layer的容器。在WMS中仅需要一个名字就能访问多个Layer。Layer Group Mode分成Single，Opaque Container，Named Tree，Container Tree，Earth Observation Tree。
