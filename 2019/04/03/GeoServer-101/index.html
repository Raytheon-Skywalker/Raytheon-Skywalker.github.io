基于Tomcat部署的GeoServer的粗浅运用

  1. 什么是GeoServer

    GeoServer is an open source software server written in Java that allows users to share and edit geospatial data. Designed for interoperability, it publishes data from any major spatial data source using open standards.
    我自己理解就是按照某种格式发布(展现)某些数据。

  2. GeoServer以及相关概念

    1. GeoServer相关概念

      1. WFS

        采用HTTP协议，通过指定的参数返回相应的要素信息，矢量数据。

      2. WCS

        该服务面向空间影像数据，它将包含地理位置值的地理空间数据作为”栅格（coverage–>raster）”在网上相互交换。

      3. WMTS

        采用HTTP协议，通过指定的参数返回相应的地图瓦片，该瓦片通常已在服务端预先按照规定格式生成。

      4. WMS

        提供基于HTTP的可被浏览器展现的地图图像数据。

      5. 其他可能感兴趣的Open Geo Spatial标准

      6. GISer必备数据格式清单

  3. GeoServer的运行基础

    1. Apache Tomcat 9

      1. 部署Tomca9

        1. 修改Server.xml中Tomcat的端口号

          术语：$apache-tomcat-9-install-root$:表示tomcat的安装位置
          位置：$apache-tomcat-9-install-root$/conf/server.xml
          Java AJP Connector:Connector port=”8009->新端口1“ protocol=”AJP/1.3”
          Java HTTP Connector:Connector port=”8080->新端口2“ protocol=”HTTP/1.1”

        2. 修改Tomcat Manager Web App 的context.xml 配置文件

          Tomcat9 对Manager App进行了保护，需要修改相关的限制来进行远程登录。我在此只是粗暴的禁用<!-- <Valve className="org.apache.catalina.valves.RemoteAddrValve" allow="127\.\d+\.\d+\.\d+|::1|0:0:0:0:0:0:0:1" /> -->
          位置：$apache-tomcat-9-install-root$/webapps/manager/META-INF/context.xml

        3. 修改Tomcat Users 配置文件

          位置：$apache-tomcat-9-install-root$/conf/tomcat-users.xml

    2. GeoServer以war包形式部署

      通过Tomcat的HTML Gui管理页面上传GeoServer的war包，并访问http://部署Tomcat服务器的IP:修改后的HTTP协议监听的port/geoserver/web/ 来验证是否已经部署成功。

  5. GeoServer的数据管理

    1. Workspace

      1. 像Namespace一样，是管理其他概念的容器。一般，相似的Layer放到同一个Workspace中，表达为WName:LayerName。相同LayerName的Layer需要放到不同的Worksapace中。

    2. Stores

      1. 链接了一个具体的数据源，可附加定制的链接参数，数据源可以是N个文件、一个文件夹或是一个数据库

    3. Layers

      1. 图层意味着一个栅格或是矢量的数据集。Layer依赖于Store中的具体数据，比如数据库中的某一个表，Layer被关联到Store所属的Workspace中。

    4. Layer Groups

      1. 是Layer的容器。在WMS中仅需要一个名字就能访问多个Layer。Layer Group Mode分成Single，Opaque Container，Named Tree，Container Tree，Earth Observation Tree。
